# OpsMind

AI-powered platform for incident investigation, root cause analysis and production troubleshooting.

## Vision

OpsMind helps support teams, developers and operations engineers quickly identify the root cause of incidents by analyzing logs, stack traces and production events.

## Features

- Incident analysis
- Root Cause Analysis (RCA)
- Log investigation
- Stack trace analysis
- Release notes generation
- Operational recommendations

## Tech Stack

- MongoDB

### Frontend
